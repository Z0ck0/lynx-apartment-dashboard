# Lynx Logo Assets

This folder contains the Lynx brand logo files used in the Streamlit app.

## Required Files

Place the following PNG logo files in this folder:

- **lynx_logo_light.png** - White logo on transparent background (for dark backgrounds)
- **lynx_logo_dark.png** - Black logo on transparent background (for light backgrounds)

## Usage in App

- **White logo (lynx_logo_light.png)** is used for:
  - Browser tab favicon (page_icon)
  - Sidebar header (dark background)
  
- **Black logo (lynx_logo_dark.png)** is used for:
  - HTML/PDF report exports (light backgrounds)
  - Any future light-themed UI elements

## File Format

- Format: PNG with transparent background
- Recommended size: 200x200px or larger (will be scaled as needed)
- Aspect ratio: Square (1:1)
